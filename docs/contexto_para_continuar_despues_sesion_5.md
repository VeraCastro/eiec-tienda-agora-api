## Estado después del Tema 5

Tema 5 quedó diseñado como sesión de 50 minutos sobre pruebas automatizadas en Tienda Ágora API.

Materiales generados:
- Guía docente de sesión.
- Documento ejecutivo de actividades previas para Moodle.
- Presentación Beamer.
- Actividad formativa no sumativa mediante Pull Request.

Decisiones pedagógicas:
- Las sesiones síncronas duran 50 minutos.
- Se usa aula virtual Moodle para preparación y trabajo posterior.
- Las actividades son estrictamente formativas.
- El trabajo puede ser individual o en equipo.
- La semana del 25 de mayo se usa como trabajo asíncrono.

Tema 6 debe continuar con:
- Calidad de código.
- Cobertura.
- Revisión de Pull Requests.
- Interpretación de reportes.
- Primeras reglas de calidad.
