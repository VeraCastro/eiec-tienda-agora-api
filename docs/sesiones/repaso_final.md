# Sesión de repaso final

## Objetivo

Recorrer el flujo completo de entrega continua usando Tienda Ágora API.

## Demo final

```text
issue
→ rama feature
→ cambio
→ pruebas
→ PR
→ pipeline
→ artefacto
→ despliegue o simulación
→ smoke test
→ rollback o feature flag
```

## Actividad por equipos

- Equipo A: agregar endpoint `/orders`.
- Equipo B: agregar prueba de contrato.
- Equipo C: romper y corregir pipeline.
- Equipo D: documentar rollback.
- Equipo E: preparar release notes.

## Evidencia final

Cada equipo entrega:

- Pull Request.
- Captura de pipeline.
- Descripción del artefacto.
- Estrategia de despliegue o rollback.
