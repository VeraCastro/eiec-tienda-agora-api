# Decisión 002: AWS como Plan A, no como dependencia

## Decisión

AWS será Plan A si el convenio institucional lo permite.

El Plan B será GitHub Actions + Docker + GitHub Container Registry + Docker Compose/local.

## Motivo

No se conoce todavía qué permite el convenio AWS.

El aprendizaje no debe depender de acceso cloud incierto.

## Consecuencia

Todas las sesiones se diseñarán para preservar el concepto de CI/CD aunque cambie el destino de despliegue.
